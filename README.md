# emoji_url
chrome extension: emoji url

emoji 图片资源取自 [http://www.emoji-cheat-sheet.com/](http://www.emoji-cheat-sheet.com/)

安装后鼠标左键点击 emoji 图标即可复制对应 emoji 表情的 url 地址。
